API Basejump: File Metadata Microservice

GitHub

Hi there! This is my implementation of FreeCodeCamp's API Basejump 'File Metadata Microservice'. This basejump utilizes Node.JS, Express, and the Multer module.

User Stories:

  1. I can submit a FormData object that includes a file upload.
  2. When I submit something, I will receive the file size in bytes within the JSON response.

Service URL:

/upload

Example usage:

1. Visit URL above
2. Choose a file and upload it
3. File size will be returned to browser
